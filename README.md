country-calling-flag
====================

this is a tool to build the json information you want

check the build.py

###run
	$ python build.py

### result
**country.json ** for your project

the default result is 

	{
	“call”:”86”,
	”ch”:”\u4e2d\u56fd”,
	“en”:”china”,
	”ja”:”\u4e2d\u56fd”
	}
	
you can change the build.py for your project
